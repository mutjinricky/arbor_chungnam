# 시연용 샘플 CSV

대시보드의 `+ 사이트 CSV` / `+ 민원 데이터` 업로드 기능을 시연·검증할 때 사용하는 가짜 데이터입니다.

## 파일 목록

### 사이트 추가용

| 파일 | 내용 | 업로드 방법 |
|---|---|---|
| **demo_streets_chungnam.csv** | 충남 12개 가로수길 (실제 공공데이터에 누락된 곡교천 은행나무길 등 포함) | 헤더 `+ 사이트 CSV` 버튼 |

포함 사이트 — 충남 가로수길 미등록 시군(아산·청양·계룡·서산) 보강:
- 곡교천 은행나무길 (아산)
- 신정호 둘레길 왕벚나무 (아산)
- 외암민속마을 보호수길 (아산)
- 청양읍 청양로 (청양)
- 칠갑산로 단풍가로수 (청양)
- 계룡시청 앞 메타세쿼이아 (계룡)
- 계룡대 진입로 가로수 (계룡)
- 해미읍성 가로수길 (서산)
- 부석사 진입로 가로수 (서산)
- 탑정호 둘레길 왕벚나무 (논산)
- 안면도 자연휴양림 진입로 (태안)
- 신두리 해안사구 가로수 (태안)

업로드 시 형식: 충청남도 가로수길정보표준데이터 (시점·종점 좌표·수종·식재년도)

### 민원 데이터 (새올·신문고 export 가정)

| 파일 | 건수 | 시군 커버리지 | 누적 효과 시연 지점 |
|---|---|---|---|
| **demo_complaints_chungnam.csv** | 50건 | 15/15 시군 | 천안 사직동 4건, 당진 송악읍 3건, 아산 곡교천 3건, 청양읍 2건 |
| sample_complaints_saeol.csv | 30건 | 14/15 (계룡 없음) | 천안 사직동 3건, 아산 곡교천 2건 |

→ `demo_complaints_chungnam.csv` 권장 (더 풍부함).

업로드 시 형식: 새올행정시스템·국민신문고 export 형식
- 컬럼: `민원ID, 시군, 주소, 위도, 경도, 접수일자, 민원유형, 처리상태, 내용`
- 좌표 기반 500m 이내 가장 가까운 사이트에 자동 매칭

## ⚠️ 정직성

이 파일들은 **시연 효과를 위해 손으로 작성한 가짜 데이터**입니다. 실제 시민이 제기한 민원·실재 공공데이터셋 등재 사이트가 아닙니다.

실 운영에선:
- 사이트 CSV → 공무원이 다운로드받은 표준데이터 또는 자체 수목 대장
- 민원 CSV → 새올·국민신문고에서 export한 실제 민원

## 시연 흐름 (권장)

1. `+ 사이트 CSV` → `demo_streets_chungnam.csv` 업로드 → 12개 추가, 검색에 "곡교천" 입력하면 표시됨
2. `+ 민원 데이터` → `demo_complaints_chungnam.csv` 업로드 → 50건 매칭
3. 우선순위 테이블 정렬 "민원 많은 순" → 천안 사직동·곡교천·당진 부곡리 상단
4. 곡교천 사이트 클릭 → 상세 패널에 "민원 3건 · 가산 +10점 · 등급 변동" 확인
5. PDF 보고서 다운로드 → A4 1~2페이지 산출
